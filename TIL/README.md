# 📝 TIL (Today I Learned)

매일 새롭게 배운 내용을 기록하는 공간입니다.

## 📌 작성 규칙

- 본인 이름으로 된 폴더 안에 작성합니다. (예: `kimssafy/`)
- 오늘 배운 것, 느낀 것, 알게 된 것을 자유롭게 기록합니다.
- 파일명은 `YYYY-MM-DD.md` 형식으로 작성합니다.
- 짧더라도 꾸준히 작성하는 것을 목표로 합니다.

## 📂 폴더 구조 예시

```
TIL/
├── kimssafy/
│   ├── 2026-02-24.md
│   └── 2026-02-25.md
├── leessafy/
│   ├── 2026-02-24.md
│   └── 2026-02-25.md
└── ...
```
