# 🎯 전체 시연 컨셉
👉 **“서류 제출부터 가심사 결과 + 근거 리포트까지 자동 생성”**

- **기존:** 사람이 서류 확인 → 규정 찾기 → 판단 → 보고서 작성
- **우리:** OCR + RAG + LLM으로 자동화

---

# 🧩 시연 시나리오 (스토리 기반)

## 1. 고객 상황 제시
> “직장인 김OO님이 주택담보대출을 신청합니다.”

- **직업:** 근로자
- **연소득:** 5,000만원
- **신용등급:** A
- **주택 보유:** 0채
- **대출 목적:** 주택 구입

## 2. 서류 업로드
> “고객이 필요한 서류를 업로드합니다.”

📂 **업로드 파일**
- 주민등록등본
- 주민등록초본
- 가족관계증명서
- 재직증명서
- 근로소득원천징수영수증
- 건강보험자격득실확인서
- 등기부등본
- 집합건축물대장
- 매매계약서

👉 **화면:** 파일 업로드 UI

## 3. OCR 엔진 동작
> “업로드된 서류는 OCR 엔진을 통해 자동으로 분석됩니다.”

**✅ 보여줄 포인트**
- 문서 자동 분류
- 필드 추출

**📝 예시:**
- **이름:** 김OO
- **소득:** 5,000만원
- **근속기간:** 3년
- **담보 주소:** 서울 OO구

## 4. AI 심사 엔진 (핵심)
> “이제 추출된 정보를 기반으로, AI가 내규를 참고하여 가심사를 진행합니다.”

💡 **핵심 강조**
- **RAG** → 관련 내규 조항 검색
- **LLM** → 조항 기반 판단

📌 **필드별 판단 (실제 시연 포인트 예시):**

1️⃣ **신용등급**
- **결과:** 승인
- **근거:** “A등급 이상은 취급 가능 (제2조)”

2️⃣ **소득 안정성**
- **결과:** 승인
- **근거:** “근속 1년 이상 + 4대보험 가입자”

3️⃣ **담보 적정성**
- **결과:** 검토
- **근거:** “해당 지역 규제지역 → LTV 제한 적용”

## 5. 계산 로직 (LTV / DSR)
- LTV 계산
- DSR 계산

**📊 예시:**
- **LTV:** 40%
- **DSR:** 40%

## 6. 최종 가심사 결과
> “모든 항목을 종합하여 가심사 결과를 도출합니다.”

- 👉 **결과:** **승인**
- **이유:** 규제지역 → 추가 심사 필요

## 7. XAI 레포트 생성
> “AI는 단순 결과가 아니라, 판단 근거를 포함한 심사 리포트를 생성합니다.”

**📄 출력 예시**
- 고객 요약
- 주요 판단 결과
- 리스크 포인트
- 참고 내규 조항
