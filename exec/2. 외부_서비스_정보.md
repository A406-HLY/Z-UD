# 외부 서비스 가입 및 활용 정보


## Cloudflare R2 (파일 스토리지)

### 서비스 개요
- **서비스 명**: Cloudflare R2 Object Storage
- **활용 목적**: 프로필 이미지, 상품 이미지, 영상 저장
- **연동 방식**: AWS S3 호환 API

### 가입 및 버킷 생성
1. [Cloudflare 대시보드](https://dash.cloudflare.com) 접속
2. 좌측 메뉴 **R2 Object Storage** 선택
3. **Create bucket** 클릭
   - 버킷 이름 예시: `zud-storage`
4. **Settings** → **Public Access** 활성화
   - Public URL 확인: `https://pub-xxxxxxxx.r2.dev/`
5. **R2 API Tokens** 생성
   - 권한: **Object Read & Write**
   - Access Key ID, Secret Access Key 저장

### 주요 설정 값

| 항목 | 환경변수명 | 설명 |
|------|-----------|------|
| Endpoint | `CLOUDFLARE_ENDPOINT` | `https://<ACCOUNT_ID>.r2.cloudflarestorage.com` |
| Bucket | `CLOUDFLARE_BUCKET` | 생성한 버킷 이름 |
| Access Key | `CLOUDFLARE_ACCESS_KEY` | R2 API Token의 Access Key ID |
| Secret Key | `CLOUDFLARE_SECRET_KEY` | R2 API Token의 Secret Access Key |
