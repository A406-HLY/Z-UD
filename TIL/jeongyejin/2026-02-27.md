## 1. 인터뷰 추가 정리 → 핵심 타겟 확정

### 🔎 실제 대출 상담 흐름 재확인

- 담보 정보 → 등기부등본으로 확인
- 신용 부채 정보 → 신용정보조회동의서 작성 후 행원이 조회
- 소득 정보 → 근로소득원천징수영수증 / 소득금액증명원
- 가심사 단계 → 고객 자가 기입 소득 기반 대략 판단
- 자서 단계 → 실제 서류 접수 후 최종 확인

### 🧠 중요한 인사이트

실제 상담 시 거의 매번 동일한 질문 반복:
- 소득 얼마인가요?
- 기존 대출 있으신가요?
- 담보 시세는 얼마인가요?
- 재직기간은 얼마나 되셨나요?

→ 고객이 비대면으로 서류 제출 시  
**자동 요약 + 핵심 정보 정리**만 되어도 업무 강도 크게 감소

### 🎯 최종 타겟 확정

**“대출 상담 시 서류 검토 시간 단축”을 메인 타겟으로 설정**

자동 승인 시스템이 아니라  
→ **가심사 보조 DSS 시스템**으로 포지셔닝
